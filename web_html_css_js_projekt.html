Általános leírás

Készíts egy honlapot! A téma tetszőleges, bármit választhatsz. A munkádra jegyet kapsz, sőt, a munkádat te magad (is) pontozod. Ehhez egy pontozólap áll rendelkezésre. Itt szerepel az is, hogy miért mennyi pont jár. Ezt a pontozólapot külön is le kell adnod, én csak ellenőrzöm, hogy jól pontoztad-e. Fontos, hogy ahol lehet, azt is megírd a pontozólapon, hogy az adott előírt feladatot hol valósítottad meg. Pl. ha az szerepel ott, hogy egy táblázat két cellája legyen összevonva, akkor írd meg azt is, hogy hol található a szóban forgó táblázat!

Az elkészült weboldalt töltsd fel valamilyen statikus weboldalak hosztolására alkalmas oldalra! Egy kényelmes lehetőség, ha ezt még nem használod másra, a GitHub Pages. A link mögött egy tutorial van, lényegében ez úgy működik, hogy a GitHub fiókodban a speciálisan elnevezett, statikus weboldalt tartalmazó repositoryt képes a github hostolni.

A repóhoz tartozó README.md fájlban hivatkozásként szerepeljen a honlap linkje is!
