Whac-a-mole

Általános leírás

A feladat a Whac-a-mole játék. Ez egy nagyon egyszerű játék. Adott egy rács, azonos színű négyzetekkel. Az egyik négyzet más színű. Erre rá kell kattintani. Aki eltalálja a kattintással a négyzetet, kap egy pontot. Amikor rákattintott az ember, visszaszíneződik a négyzet színe ugyanarra, mint a többi. Viszont egy másik átvált arra a színre, amire rá kell kattintani. És így tovább.

Hozz létre egy oldalt, ahol a felhasználó mezőkre kattintgathat, amik a fenti szabályokat követik.

Az oldal legyen szép, esztétikus. Állíts be háttérszínt, headert, legyen h1-es elem is a lapon (akár maga a header szövege). A kockák közt legyen hézag, a kockákon legyen árnyékolás, kis lekerekített határvonal. A kockák fölött elhúzva az egeret, következzen be valami effektus is. A kockákra kattintva is következzen be valamilyen effektus, lehetőleg valamiféle animáció-szerű effektus.

Vezess egy számlálót is, hogy hány pontnál tartasz. A cél természetesen az lesz, hogy minél több pontot szerezzen az ember egy adott idő alatt...

Az elkészült weboldalt töltsd fel valamilyen statikus weboldalak hosztolására alkalmas oldalra! Egy kényelmes lehetőség, ha ezt még nem használod másra, a GitHub Pages. A link mögött egy tutorial van, lényegében ez úgy működik, hogy a GitHub fiókodban a speciálisan elnevezett, statikus weboldalt tartalmazó repositoryt képes a github hostolni.

A repóhoz tartozó README.md fájlban hivatkozásként szerepeljen a honlap linkje is!
